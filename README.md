# 💖 Love Proposal Web Page

A fun and interactive **Love Proposal Website** built using **HTML, CSS, and JavaScript**.

## ✨ Features

* 💕 Beautiful romantic UI
* ❤️ Animated heart effect
* 😂 Funny "No" button that runs away when hovered
* 🎥 Hidden surprise video that plays after clicking **Yes**
* 📱 Responsive and simple design
* ⚡ Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```
Love-Proposal/
│── index.html
│── style.css
│── love.mp4.mp4
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.
4. Click **Yes** to watch the surprise video.
5. Try clicking **No** and see what happens! 😄

## 📸 Preview

A cute and interactive proposal page with a romantic theme, animated elements, and a funny escaping **No** button.

## 🎯 Purpose

This project was created to practice JavaScript DOM manipulation, event handling, animations, and basic web development concepts in a fun way.
